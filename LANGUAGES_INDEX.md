## Java

IntelliJ IDEA's primary language support with extensive features for Java development.

**Key Features:**
- Code generation and refactoring shortcuts
- JUnit testing integration
- Maven/Gradle build system support
- Spring Framework integration
- Debugging tools

**Common Shortcuts:**
- `Alt + Insert` - Generate code (getters, setters, constructors)
- `Ctrl + Alt + T` - Surround with template
- `Ctrl + Shift + U` - Toggle case
- `Ctrl + Alt + O` - Optimize imports

## Universal Shortcuts (All Languages)

These shortcuts work across all language environments in IntelliJ IDEA:

| Shortcut | Action |
|----------|--------|
| `Ctrl + Space` | Code completion |
| `Ctrl + B` | Go to definition |
| `Ctrl + Shift + B` | Go to type definition |
| `Alt + F7` | Find usages |
| `Ctrl + H` | Show hierarchy |
| `Ctrl + F` | Find in file |
| `Ctrl + H` | Find and replace |
| `Ctrl + Alt + L` | Reformat code |
| `Ctrl + /` | Comment/uncomment |
| `F11` | Toggle bookmark |
| `F2` | Navigate to next error |
| `Shift + F2` | Navigate to previous error |
| `Ctrl + F12` | File structure |
| `Alt + Home` | Show file navigation bar |
