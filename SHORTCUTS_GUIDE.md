## üìù Tips for Learning Shortcuts

1. **Enable Key Promoter X Plugin** - Shows shortcuts while you use the mouse
2. **Customize in Settings ‚Üí Keymap** - Adjust shortcuts to your preference
3. **Vim Keybindings** - Available if you prefer Vim mode
4. **Print the PDF** - Download official keyboard shortcuts PDF
5. **Practice Regularly** - Use shortcuts daily to build muscle memory

## üîß How to Customize Shortcuts

1. Go to **File ‚Üí Settings (Windows/Linux) or Preferences (Mac)**
2. Navigate to **Keymap**
3. Search for the action you want to customize
4. Right-click and select **Add Keyboard Shortcut**
5. Press the keys you want to use
6. Click **OK**
